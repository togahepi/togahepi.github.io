Learning List

HTML
HTML là chữ viết tắt của cụm từ Hypertext Markup Language (ngôn ngữ đánh dấu siêu văn bản).
CSS
CSS là viết tắt của cụm từ Cascading Style Sheet, nó là một ngôn ngữ quy định cách hiển thị của các thẻ html trên trang web.
Javascript
Javascript là một ngôn ngữ lập trình kịch bản dựa trên đối tượng được phát triển từ các ý niệm nguyên mẫu.
Javascript là một ngôn ngữ kì lạ ngắn gọn đến kì lạ
